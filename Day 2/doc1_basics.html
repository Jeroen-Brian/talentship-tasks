HTML: Introduction and Basics

HTML is a markup language used by documents which are designed to be displayed in web browser.
It was created by Tim Burners-Lee in 1989 and later went into further development in the late 1990's.

Elements of a HTML document:

  1. Under 'head' element:
    • 'title'
    • 'link'
  2. Under 'body' element:
    • 'h1,h2,...h6'
    • 'p'
    • 'a'
    • 'div'
    • 'span'
    • 'form'
    • 'table'
